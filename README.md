# Scripts

## mgames

```
usage: mgames [-h] [--fout FOUT] [--home-only] start end

Query MLB API to get Mariners schedule from given date range

positional arguments:
  start        Search start date (most formats should work)
  end          Search end date

optional arguments:
  -h, --help   show this help message and exit
  --fout FOUT  Export result to csv
  --home-only  Show only home games
```
